# Salt & Smoke

Salt & Smoke is a premium restaurant landing page for a smokehouse dining concept in Watford. It showcases the brand story, menu, reservation flow, chef team, contact details, and newsletter signup in a polished single-page experience.

## Live site

- Live website: https://hamid-aa80.github.io/Salt---Smoke/
- Repository: https://github.com/Hamid-aa80/Salt---Smoke.git

## Overview

The site includes:

- A hero section with a booking call to action
- Service highlights for chef quality, food, ordering, and support
- An about section with brand storytelling and imagery
- A menu section with category filters, search, and a daily chef's pick
- A reservation form with validation, draft saving, and confirmation handling
- A chef team section
- A contact footer with address, opening hours, social links, and newsletter signup

## Features

- Responsive navigation with smooth scrolling
- Sticky navbar on scroll
- Animated sections using WOW.js
- Menu filtering by breakfast, lunch, dinner, and drinks
- Menu search with live results and empty-state handling
- Reservation form validation with localStorage and sessionStorage support
- Newsletter signup validation with saved email
- Dynamic opening-hours status
- Back-to-top button
- Reduced-motion support

## Tech stack

- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3
- Font Awesome
- jQuery
- WOW.js
- Google Fonts

## Project structure

- `index.html` - main Salt & Smoke homepage
- `submit.html` - reservation confirmation page
- `style.css` - site styling and theme
- `main.js` - interactive behaviour
- `assets/img/` - imagery and icons

## Local development

1. Clone the repository:
   ```bash
   git clone https://github.com/Hamid-aa80/Salt---Smoke.git
   ```
2. Open `index.html` in your browser, or serve the folder with a simple local server.

## Deployment

This project is deployed with GitHub Pages from the `main` branch.
