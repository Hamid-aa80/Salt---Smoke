📊 DATABASE SCHEMA
==================

RESERVATIONS table
├── id              INTEGER PRIMARY KEY
├── name            TEXT (required, 2+ chars)
├── email           TEXT (required, valid format)
├── date            TEXT (required, YYYY-MM-DD)
├── time            TEXT (required, HH:MM)
├── guests          INTEGER (required, 1-20)
├── requests        TEXT (optional)
├── created_at      DATETIME (auto)
└── status          TEXT (confirmed)

NEWSLETTER_SIGNUPS table
├── id              INTEGER PRIMARY KEY
├── email           TEXT (unique, required)
├── created_at      DATETIME (auto)
└── status          TEXT (subscribed)

MENU_ITEMS table
├── id              INTEGER PRIMARY KEY
├── name            TEXT (required)
├── category        TEXT (required)
├── description     TEXT (optional)
├── price           REAL (optional)
├── image           TEXT (optional)
├── is_chefs_pick   INTEGER (0/1)
└── created_at      DATETIME (auto)


🔌 API ENDPOINTS SUMMARY
========================

HEALTH CHECK
  GET /api/health
  → {"status": "API is running"}

RESERVATIONS
  POST   /api/reservations             Create reservation
  GET    /api/reservations             List all
  GET    /api/reservations/:id         Get one

NEWSLETTER
  POST   /api/newsletter/signup        Subscribe
  GET    /api/newsletter/signups       Get all

MENU
  GET    /api/menu                     Get all items
  GET    /api/menu?category=lunch      Filter by category
  GET    /api/menu/:id                 Get one item
  POST   /api/menu                     Create item
  PUT    /api/menu/:id                 Update item
  DELETE /api/menu/:id                 Delete item


📝 QUICK REFERENCE
===================

Start server:
  npm start

Run tests:
  bash test-api.sh

Test specific endpoint:
  curl http://localhost:5000/api/health

Test reservation creation:
  curl -X POST http://localhost:5000/api/reservations \
    -H "Content-Type: application/json" \
    -d '{
      "name":"John Doe",
      "email":"john@example.com",
      "date":"2024-12-25",
      "time":"19:30",
      "guests":4
    }'

Get menu items:
  curl http://localhost:5000/api/menu

Filter menu by category:
  curl http://localhost:5000/api/menu?category=lunch
